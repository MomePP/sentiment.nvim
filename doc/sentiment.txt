INSTALLATION                                          *sentiment-installation*

**NOTE**: Keep in mind that calling `setup` disables the built-in
matchparen.vim
<https://github.com/neovim/neovim/blob/master/runtime/plugin/matchparen.vim>
plugin.


- lazy.nvim <https://github.com/folke/lazy.nvim>
    >
        {
          "utilyre/sentiment.nvim",
          name = "sentiment",
          version = "*",
          opts = {
            -- config
          },
        }
    <
- packer.nvim <https://github.com/wbthomason/packer.nvim>
    >
        {
          "utilyre/sentiment.nvim",
          tag = "*",
          config = function()
            require("sentiment").setup({
              -- config
            })
          end,
        }
    <


USAGE                                                        *sentiment-usage*


- `:DoMatchParen`
    Disable the plugin.
- `:NoMatchParen`
    Re-enable the plugin.
